# Skill: Investigación Profunda

Este skill proporciona técnicas para investigaciones exhaustivas y compilación de información de calidad.

## Estrategias de Búsqueda

### Búsqueda en Capas
1. **Capa 1 - Panorama general**: "[tema] overview 2024"
2. **Capa 2 - Datos específicos**: "[tema] statistics data research"
3. **Capa 3 - Expertos/Fuentes**: "[tema] expert analysis report"

### Queries Efectivas

| Objetivo | Patrón de Query |
|----------|-----------------|
| Estadísticas | "[tema] statistics 2024 2025" |
| Tendencias | "[tema] trends forecast" |
| Comparativas | "[tema] vs [alternativa] comparison" |
| Casos de éxito | "[tema] case study success" |
| Problemas | "[tema] challenges issues" |

## Compilación de Resultados

### Estructura de Reporte

```markdown
## Resumen Ejecutivo
[2-3 puntos clave]

## Datos Encontrados
- **Estadística 1**: [valor] - [fuente]
- **Estadística 2**: [valor] - [fuente]

## Tendencias Identificadas
1. [Tendencia 1]
2. [Tendencia 2]

## Fuentes Consultadas
- [URL 1]: [descripción]
- [URL 2]: [descripción]
```

## Validación de Fuentes

### Criterios de Calidad
- Fecha reciente (preferir últimos 12 meses)
- Fuente reconocida (instituciones, empresas, medios)
- Datos citados con metodología
- Múltiples fuentes que confirmen el dato

### Red Flags
- Datos sin fecha
- Fuentes anónimas
- Afirmaciones sin respaldo
- Conflictos de interés obvios
