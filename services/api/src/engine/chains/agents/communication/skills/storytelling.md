# Skill: Storytelling y Narrativa

Este skill proporciona técnicas para crear narrativas efectivas y comunicación persuasiva.

## Estructuras Narrativas

### Problema → Solución → Beneficio
1. **Problema**: Describe el dolor/desafío actual
2. **Solución**: Presenta la respuesta
3. **Beneficio**: Muestra el resultado positivo

### Viaje del Héroe (simplificado)
1. **Situación inicial**: Contexto y normalidad
2. **Desafío**: Algo rompe el equilibrio
3. **Transformación**: Proceso de cambio
4. **Nuevo estado**: Mejor que antes

### AIDA (Atención → Interés → Deseo → Acción)
- **A**tención: Hook impactante
- **I**nterés: Datos relevantes
- **D**eseo: Beneficios emocionales
- **A**cción: Call-to-action claro

## Tonos de Comunicación

| Tono | Cuándo Usar | Características |
|------|-------------|-----------------|
| Formal | Corporativo, institucional | Objetivo, datos, sin coloquialismos |
| Cercano | Startups, B2C | Tú, ejemplos cotidianos, humor |
| Inspirador | Liderazgo, visión | Emocional, futuro, posibilidades |
| Técnico | Especialistas | Jerga, precisión, profundidad |
| Urgente | Crisis, ofertas | Directo, fechas, consecuencias |

## Elementos de Impacto

### Hooks que Funcionan
- Pregunta provocadora: "¿Sabías que...?"
- Estadística sorprendente: "El 78% de..."
- Contradicción: "Lo que todos creen está mal"
- Historia personal: "Hace 3 años, yo..."

### Mensajes Clave
- Máximo 3 ideas principales por comunicación
- Repetir el mensaje de formas diferentes
- Anclar con datos memorables
- Cerrar con call-to-action específico

## Arcos Emocionales

### Para Presentaciones
```
Curiosidad → Comprensión → Sorpresa → Convicción → Motivación
```

### Para Propuestas
```
Empatía (entiendo tu problema) → Credibilidad (tengo experiencia) → Esperanza (hay solución) → Confianza (puedo ayudarte)
```
