# Skill: Branding e Identidad Visual

Este skill proporciona conocimiento para crear logos, paletas de colores y elementos de identidad visual.

## Principios de Diseño de Logos

### Simplicidad
- Un buen logo funciona en cualquier tamaño (desde favicon hasta valla publicitaria)
- Máximo 2-3 elementos visuales
- Formas geométricas básicas son más memorables

### Versatilidad
- Debe funcionar en positivo y negativo
- Colores sólidos y versión monocromática
- Horizontal, vertical y cuadrado (si aplica)

### Relevancia
- Refleja la industria o valores de la marca
- Evita clichés obvios (bombilla para "ideas", etc.)
- Atemporal > trendy

## Prompts Efectivos para Generación de Logos

### Estructura de un buen prompt
```
[Estilo] logo for [Tipo de negocio], [Elementos visuales], [Colores], [Características adicionales]
```

### Ejemplos de prompts efectivos

**Tech/SaaS:**
```
Minimal geometric logo for AI startup, abstract neural network symbol, gradient blue to purple, clean lines, modern, vector style, white background
```

**Consultoría:**
```
Professional monogram logo letters "AB", elegant serif typography, gold and navy blue, premium feel, minimalist, corporate identity
```

**Eco/Sostenibilidad:**
```
Organic leaf-based logo for sustainable brand, flowing curves, earth tones green and brown, natural feel, hand-drawn style, eco-friendly
```

**Fintech:**
```
Abstract fintech logo, upward arrow integrated with coin symbol, gradient teal to blue, trustworthy, modern, flat design
```

**Salud/Wellness:**
```
Wellness brand logo, abstract human figure in meditation pose, soft gradient pink to lavender, calming, balanced, spa aesthetic
```

## Tipografías Recomendadas

### Sans-serif Modernas
- **Inter**: Versátil, excelente legibilidad
- **Poppins**: Amigable, geométrica
- **Montserrat**: Elegante, profesional
- **Nunito**: Suave, accesible

### Serif Elegantes
- **Playfair Display**: Lujo, editorial
- **Merriweather**: Clásica, legible
- **Lora**: Equilibrada, contemporánea

### Display/Títulos
- **Bebas Neue**: Impactante, bold
- **Oswald**: Condensada, moderna
- **Archivo Black**: Fuerte, statement
